个人随笔-五月份

  • 个人感悟

时间过得相当快，转眼间离开校园到北京工作两个月了，渐渐的熟悉了北京的生活，没有了当初的青涩和不安。怎么说呢，对于工作的生活和学校生活差别还是蛮大的，看似每天工作强度没有想象的那么大，可还是每天忙忙碌碌的，比学校的节奏要快很多。每天早上7点多起床，匆匆忙忙的收拾这收拾那，8点多钟骑车去上班，当然现在还是蛮享受骑车的感觉的，每天上下班骑行30公里，大约来回需要2个小时，既可以锻炼身体也可以磨练自己的意志，看似简单的东西，如果想要坚持也是比较困难的，当然对于我们这种刚毕业的屌丝学生来说，更重要的原因就是可以省下一天10块钱的地铁费，这样算下来一月也是不小的费用。。。哈哈。。。至少够出去high一顿了~~
当然每天北京早上的交通还是无力吐槽了。。在去公司的路上至少有4处拥堵，这种拥堵已经干扰到骑车了。。。不用说也能够想象的到这种堵车程度的惨烈。。好在两轮可以上人行道。。嗷嗷的跑。。现在北京的天气还是比较适合骑行的，早晚的温度还是需要穿长袖的，穿上朕的速干衣，基本身上就不会有汗，挺舒服的。。瞬间联想到呆了6年的大济南。。。别的不说了。。在学校每年这时候已经凉水嗷嗷洗澡了。。那酸爽。。。
最近在公司忙着搞漏扫系统的一键部署工作，虽然之前在学校听师兄说要给北邮搞一套。。可是一直并没有进行。。这下好了。。欠下的孽债需要还了。。虽然说这只是个体力活，可是对于从未正经写过shell脚本的屌丝来说，还是费了一些力气，用了一周的时间才完成系统前端的部署。。不过看着自己写的几百行脚本，嗷嗷的跑。。打印这各种信息。。然后。。就没有然后了。。按照进度要求，这个月底需要完成一键部署的工作。。现在看来。。下周差不多能够搞定吧。。等把任务搞定了。。一定要好好学学shell，对于一个linux程序猿。。不能熟练掌握shell。。其实是很苦逼的。。经过这周的摸索，真正体会到了shell的强大。。其实细细想想，这些尼玛都该在学校搞定的。。只怪自己当时太放荡。。其实最近还有很多东西没有整理，，当然这些都是学校欠下的孽债。。这样看来自己在学校的温室里并没有什么努力的迹象。。为了防止忘记整理，在这里先列一下吧。。。

  1. 整理Makefile的东东，这东西可以说是linux程序猿必备的技能包。。可是自己却一直因为某种原因抵触。。啥也不说了。。只能说IDE坑死英雄好汉。。
  2. 整理linux链接库，编译，安装的东东，这东西最近一直在搞。。因为懒惰。。
  3. 整理GNU Autotools的东东，这东西就是目前linux下大型软件比较通用的打包方式，就是传说中的 ./configure make make install 的形式。
  4. 整理Cmake的东东，貌似比GNU Autotools更先进的打包方式。。当然自己没啥感觉。。后续编译可能会用到。。
  5. 阅读《程序猿的自我修养》这本书，目前看完了第一章了。。真的很好。。恶补了很多东西。。
  6. 阅读《APUE》。。不用说了。。必读的。。其实早就该看了。。
  7. 整理shell的东东。。至少能够搞定手头的工作需要。。

额。。好吧。。只能说。。无力吐槽。。孽债太多。。
